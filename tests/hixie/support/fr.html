Prérequis
Les navigateurs qui sont soumis à ce test devrait soutenir HTML2.

1. LINK éléments de la structure

Ce document contient les éléments suivants LINKs:

<!--1-->  <LINK title="La page en n&eaute;erlandais"
		type="text/html"
		rel="alternate"
		hreflang="nl"
		href="placeholder.html">

<!--2-->  <LINK title="Le manuel in Portuguese"
		type="text/html"
		rel="alternate"
		hreflang="pt"
		href="pt.html">

<!--3-->  <LINK title="Le manuel en arabe"
		type="text/html"
		rel="alternate"
		dir="rtl"
		charset="ISO-8859-6"
		hreflang="ar"
		href="placeholder.html">

<!--4-->  <LINK lang="fr" title="La documentation en Français"
		type="text/html" 
		rel="alternate"
		hreflang="fr"
		href="fr.html">

<!--5-->  <LINK title="Le manuel pour l’impression"
		type="text/html"
		rel="alternate"
		media="print"
		href="placeholder.html">

UAs (navigateurs Web) peut soutenir le LINK élément, et peut donc en quelque sorte donner accès à ces documents. Ce n’est pas requis par la spécification HTML 4.0. La spécification dit que c’est une option.

Les documents en question sont les suivants:

1. Le manuel en néerlandais
Version alternative de ce document, mais en néerlandais.
2. Le manuel en portugais
Version alternative de ce document, mais en portugais.
3. Le manuel en arabe
Version alternative de ce document, mais en arabe.
4. La documentation en Français
Version alternative de ce document, mais en français.
5. Le manuel pour l’impression
Version alternative de ce document, mais pour la presse écrite. (Au lieu de permettre un accès direct à ce document, le navigateur peut simplement l’utiliser lors de l’impression.)

Remarque. La langue du document lié à chaque fois n’est pas dans le langues de ce document prétend qu’il est. Votre navigateur peut légitimement se plaindre à ce sujet.
